ALGORITMA PSUDECODE Menghitung Nilai Nilai Akhir Siswa

**Judul**
Menghitung Nilai Nilai Akhir Siswa

Memulai program:

**Deklarasi**
-NamaSiswa : string
-NilaiKeaktifan : integer
-NilaiTugas : integer
-NilaiUjian : integer
-NilaiAkhir : float

**Input**
-NamaSiswa
-NilaiKeaktifan
-NilaiTugas
-NilaiUjian

**Proses**
-NilaiMurniKeaktifan = NilaiKeaktifan * 0.2
-NilaiMurniTugas = NilaiTugas * 0.3
-NilaiMurniUjian = NilaiUjian * 0.5
-NilaiAkhir = (NilaiMurniKeaktifan + NilaiMurniTugas + NilaiMurniUjiam)

**Output/Hasil**
-NamaSiswa
-NilaiMurniKeaktifan 
-NilaiMurniTugas 
-NilaiMurniUjian
-NilaiAkhir
