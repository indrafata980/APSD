ALGORITMA PSUDECODE Menghitung Nilai Rata-rata Siswa


**Judul**
Menghitung Nilai Rata-rata Siswa

Memulai program:

**Deklarasi**
-NIM : integer
-NamaSiswa : string
-NilaiTugasI: integer
-NilaiTugasII : integer
-NilaiTugasIII : integer
-NilaiRata_rata: float

**Input**
NamaSiswa
-NilaiTugasI
-NilaiTugasII
-NilaiTugasIII

**Proses**
NilaiRata_rata = (NilaiTugasI + NilaiTugasII + NilaiTugasIII) / 3

**Output/Hasil**
-NIM
-NamaSiswa
-NilaiRata_rata
