> Membuat sebuah program menghitung nilai rata-rata siswa.

Nama Siswa : string
NIM Siswa : integer
Nilai Tugas I : integer
Nilai Tugas II : integer
Nilai Tugas III : integer
Nilai Rata-rata : integer

read(NIM Siswa, Nama Siswa, Nilai Tugas I, Nilai Tugas II, Nilai Tugas III, Nilai Rata-rata)
Nilai Rata-rata = (Nilai Tugas 1 + Nilai Tugas II + Nilai Tugas III) /3
write(NIM Siswa, Nama Siswa, Nilai Rata-rata)
end
